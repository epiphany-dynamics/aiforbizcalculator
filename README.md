# AI for Business Calculator

A free ROI calculator that estimates the time and money a service business would save by automating its highest-leverage workflow. Built and run by [Epiphany Dynamics](https://epiphanydynamics.ai) as a public lead-generation tool.

**Live:** [ai4bizcalculator.online](https://ai4bizcalculator.online)

## Stack

Astro, React, TypeScript, Tailwind CSS — deployed on Vercel via GitHub Actions.

## What it does

Walks a visitor through a short scenario input (industry, current manual process, volume), then estimates hours saved and dollar value of automating that workflow. Output routes into the agency's lead pipeline.
